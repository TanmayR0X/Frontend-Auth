### üîê Frontend Authentication System (Login & Signup)

A pure frontend authentication system built using HTML, CSS, and JavaScript that allows users to sign up, log in, maintain sessions using localStorage, and access a protected dashboard with session expiry logic.

This project simulates real-world authentication flow without backend integration and is ideal for learning client-side auth concepts.

### ‚ú® Features

| Feature | Description |
|------|-------------|
| Signup | User can register with name, email and password |
| Login | User can login with registered credentials |
| Validation | Email regex and password length validation |
| Duplicate Check | Prevents signup with existing email |
| Dashboard | Displays personalized greeting |
| Session Handling | Uses localStorage for login session |
| Logout | Clears session and redirects to login |
| Protected Routes | Blocks dashboard without login |
| Tooltip Feedback | Global notification messages |
